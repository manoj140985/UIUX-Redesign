# 🎵 Obsedian Music – UI/UX Redesign

A modern UI/UX redesign of the **Obsedian Music** streaming platform focused on creating an immersive, intuitive, and visually engaging listening experience.

---

## 📖 Project Overview

The goal of this project is to redesign the Obsedian Music platform using modern UI/UX principles. The redesign enhances navigation, improves music discovery, simplifies playlist management, and creates a premium interface while maintaining a consistent user experience.

---

## 🎯 Objectives

- Create a clean and modern user interface.
- Improve overall user experience.
- Enhance navigation and accessibility.
- Simplify music discovery and playback.
- Design a consistent and responsive interface.
- Improve visual hierarchy and readability.
